Dette skriptet logger seg på e-post, leser avsendere og legger avsendere inn i en fil som heter avsendere.csv, her er det flere kollonner, den siste kollonnene skal være ja eller nei.  Default er nei, hvis dette endres til ja, vil e-posten bli flytt til mappen MAS.  Man må først endet filen env.org.txt til .env og sette inn riktige verdier der.  Når dette er gjort kan man kjøre flytt-epost.go, da blir e-post flyttet.
